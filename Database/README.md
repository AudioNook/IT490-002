### Database
This directory contains code for database interaction in the AudioNook application. It includes the config.php file which contains the database configuration and relevant classes for DB.

```
├── database/
│   ├── lib/
│   │   ├── config.php
│   │   └── db.php, .env, etc.
│   ├── sql/
│   │   ├── .sql files
│   │   └── init_db.php
│   ├── logs/
│   └── db_listner.php
```
